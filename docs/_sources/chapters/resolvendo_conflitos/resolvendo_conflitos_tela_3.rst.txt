Nenhum conflito detectado
=========================

Se nenhuma das duas telas anteriores se apresentar, quer dizer que com o ``git pull``, o git conseguiu com sucesso fazer
a integração (``merge``) entre os códigos-fonte do repositório remoto e repositório local.

Podemos prosseguir normalmente a partir de então, fazendo um ``git push origin main``. Com isso, o código-fonte do
repositório local, atualizado às modificações anteriores do repositório remoto, estará no repositório remoto:
